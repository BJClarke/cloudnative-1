# SDK
{: #sdk}

{{site.data.keyword.Bluemix}} Web サービス SDK およびモバイル・サービス SDK をアプリに追加するには、使用する SDK を選択し、それらの SDK をアプリにプルするように依存関係マネージャーを構成します。
