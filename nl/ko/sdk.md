# SDK
{: #sdk}

앱에 {{site.data.keyword.Bluemix}} 웹 및 모바일 서비스 SDK를 추가하려면 사용할 SDK를 선택하고 사용자의 앱으로 SDK를 가져오도록 종속 항목 관리자를 구성하십시오.
